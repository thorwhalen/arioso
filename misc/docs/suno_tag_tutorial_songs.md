# Suno Tag Tutorial Songs

> **How these work:** Each song uses a **narrator voice** (`[Spoken Word]`) to name and explain each tag, followed by a `(bell ding)` audio cue, then the tag activates so you can *hear* its effect. The contrast between the narrator explaining and the tag kicking in is the educational payload.
>
> **How to use:** Copy-paste each "LYRICS" block directly into Suno's lyrics field. Put the "STYLE" text into Suno's style prompt field. Generate, listen, learn.

## Tips for Best Results

1. **Generate each song 2–3 times** — Suno is stochastic; some generations will nail the tag transitions better than others
2. **If a tag doesn't take effect**, try placing it on its own line before the lyrics it should affect
3. **The `(bell ding)` cue** is a parenthetical stage direction — Suno often renders it as a small percussive sound or transition. If it doesn't work on your generation, try `(ding)` or `(chime)` instead
4. **Song length**: Each of these is designed for Suno's standard ~3-minute generation
5. **Style prompt is crucial**: The style prompt sets the overall palette; the in-lyrics tags then modify within that palette
