arioso.base

Core types and constants for arioso.

Defines the unified output types (Song, AudioResult) and the canonical set of 40 affordance names that all platform adapters map to.

class arioso.base.AudioResult(audio_bytes: bytes | None = None, audio_url: str = '', audio_array: Any | None = None, sample_rate: int = 0, format: str = '', duration_seconds: float = 0.0)[source]

Unified audio output from any music generation platform.

Exactly one of audio_bytes, audio_url, or audio_array will typically be populated by the platform adapter. Conversion between forms can be done on demand by the caller.

audio_bytes

Raw audio file bytes (MP3, WAV, etc.).

audio_url

Type:

audio_array
