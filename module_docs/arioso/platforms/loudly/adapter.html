arioso.platforms.loudly.adapter

Loudly adapter via REST API.

class arioso.platforms.loudly.adapter.Adapter(config: dict)[source]

Adapter for Loudly music generation.

Loudly may return the audio URL directly or require polling, depending on the generation complexity.

generate(prompt: str, *, duration: float = 30, genre: str = '', bpm: int | None = None, key: str = '', energy: float | None = None, instruments: list[str] | None = None, structure: list | None = None, wait_for_completion: bool = True, poll_interval: float = 5.0, timeout: float = 300.0, **kwargs) → list[Song][source]

Generate music via Loudly.

Parameters:

  • prompt – Text description of desired music.

  • duration – Track duration in seconds.

  • genre – Genre tag (50+ options available).

  • bpm – Tempo in beats per minute.

  • key – Musical key (e.g. ‘C major’).

  • energy – Energy level (0-1).

  • instruments – List of instruments (up to 7).

  • structure – Song section structure.

  • wait_for_completion – If True, poll until audio is ready.

  • poll_interval – Seconds between status checks.

  • timeout – Max seconds to wait for completion.

Returns:

List containing a single Song object.
