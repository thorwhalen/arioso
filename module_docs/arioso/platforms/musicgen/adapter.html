arioso.platforms.musicgen.adapter

MusicGen adapter using audiocraft (preferred) or transformers.

class arioso.platforms.musicgen.adapter.Adapter(config: dict)[source]

MusicGen adapter with lazy model loading.

Tries audiocraft first; if unavailable, falls back to the HuggingFace transformers pipeline.

generate(prompt: str, *, duration: float = 8.0, temperature: float = 1.0, top_k: int = 250, top_p: float = 0.0, guidance: float = 3.0, model: str = 'facebook/musicgen-small', **kwargs) → Song[source]

Generate music from a text prompt.

Parameters:

  • prompt – Text description of desired music.

  • duration – Length in seconds.

  • temperature – Sampling randomness.

  • top_k – Top-k sampling parameter.

  • top_p – Top-p nucleus sampling.

  • guidance – Classifier-free guidance scale.

  • model – Model variant name (e.g. ‘facebook/musicgen-small’).

Returns:

A Song with audio_array populated.
